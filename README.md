# Prompt Craft — Learn to Prompt AI

A user-friendly, animated website for learning how to prompt AI tools (ChatGPT, Claude, etc.).

## Run locally

Open `index.html` in your browser, or use a simple server:

```bash
# Python 3
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Contents

- **Basics** — Be specific, set format/length, define audience, add constraints
- **Techniques** — Role-playing, few-shot examples, chain of thought, negative instructions
- **Before & after** — Weak vs strong prompt examples
- **Pro tips** — Iterate, break tasks down, add context, one thing at a time

## Files

- `index.html` — Structure and content
- `styles.css` — Layout, design, and animations
- `script.js` — Scroll animations, mobile menu
